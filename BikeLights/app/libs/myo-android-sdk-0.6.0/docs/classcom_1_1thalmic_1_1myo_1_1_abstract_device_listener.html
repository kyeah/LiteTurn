Detailed Description

This class provides empty implementations of the methods from DeviceListener.

Any listener that cares only about a subset of the methods can simply subclass this class instead of implementing the interface directly.

Examples:
BackgroundService.java, HelloWorldActivity.java, MultipleMyosActivity.java, and MyoGlassService.java.
